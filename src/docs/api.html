Knurl Mastering API & Agent Integration Docs

Agent Ready

Welcome! Knurl Mastering is fully compatible with modern agent discovery, capability mapping, and real-time interactive browser tools.

1. RFC 9727 API Discovery

We publish our standardized API Catalog at the well-known path: /.well-known/api-catalog.

This allows crawlers and agents to discover our core services, OpenAPI endpoints, and health interfaces natively.

2. OpenAPI Description

The machine-readable OpenAPI 3.0 specification is available here: /api/openapi.json.

3. WebMCP Site Tools

If you are viewing this page in an agent-enabled browser (supporting the Web Model Context Protocol), the following WebMCP tools are registered dynamically on our homepage:

  • getStudioInfo: Retrieve biography, staff details, and external portfolios.
  • getServices: List stereomastering, stem mastering, vinyl preparation capabilities and terms.
  • submitInquiry: Helper tool to format and stage service inquiries.
