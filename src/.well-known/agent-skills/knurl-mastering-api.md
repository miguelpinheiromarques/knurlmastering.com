# Knurl Mastering Agent Skill

Guidelines and capabilities for AI agents interacting with the Knurl Mastering portfolio, credit catalogs, and engineers.

## Description
This skill enables agents to easily query professional audio mastering studio information, services, portfolios for Miguel Pinheiro Marques and Clara Araújo, and submit reservation or inquiry details.

## Capabilities
- Explore stereo mastering, stem mastering, and vinyl pre-mastering services.
- Retrieve official contact emails and studio location properties.
- Automate form fill parameters for user inquiries.
